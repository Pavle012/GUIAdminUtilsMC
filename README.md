# Admin Utils GUI Datapack

Admin Utils GUI is a Minecraft datapack that adds powerful, easy-to-use graphical interfaces for server administrators.
All menus are built using Minecraft dialogs, meaning no chat spam, no commands to remember â€” just clean GUIs.

> IMPORTANT
> 
> Requires Minecraft 1.21.6 or newer (dialogs are not available in older versions).




---

Features:

- GUI-based admin tools

- Clean menus instead of long commands

- Designed for multiplayer servers

- Fast and lightweight (pure datapack)



---

## How It Works

This datapack uses Minecraft dialogs to display interactive menus.
Admins can open GUIs and perform actions without typing complex commands, making moderation and server management faster and safer.


---

## Usage

Access the admin GUI using the provided command or trigger

Navigate menus using buttons

All actions run safely in the background
